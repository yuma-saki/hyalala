特集 ひゃらら、とは何者か？　緊急解説

「野球経験より卓球経験の方が長いチームです」——本人たちが堂々と宣言

沖縄県中頭郡西原町。この静かな町から、前代未聞の草野球チームが誕生した。チーム名は「ひゃらら」。そのキャッチコピーは「西原町発・99世代・全員元卓球部」。

全員が1999年生まれで構成されており、驚くべきことに野球経験者は少数派。メンバーの多くは卓球部出身であり、ラケットを握った手でバットを振るという、常識を覆すチームである。

「フォームだけはプロ並みですが、打率はお察しください」とあるメンバーは苦笑いを浮かべながら語った。しかし誰一人として諦めていない。それがひゃらら流だ。

独占告白：「サーブの回転は完璧、でも外角低めはお手上げ」

取材を進めると、衝撃の実態が浮かび上がってきた。メンバー全員が卓球のサーブ技術には自信を持つ一方、野球の基本スキルは「これから」という状況だという。

チームのスローガンは、笑い率が打率を超えること。試合中に笑いが起きれば、それは勝利に等しい——そんな哲学がチームを一つにまとめている。

「気合いだけは誰にも負けない」とメンバーB。「根性さえあれば何でもできる」とメンバーF。その言葉に嘘はない。少なくとも、本人たちは本気だ。

緊急速報 チームデータ　4つの衝撃事実
DATA 01
創設世代（99年・平成生まれ）
1999年生まれで結成。令和の草野球界に平成のノリを持ち込む異色の存在として注目を集めている。
DATA 02
全員卓球経験あり（野球経験者は少数派）
チームの大半は卓球部出身。野球経験者は希少種。これが強みになるか弱点になるか、それは試合を見てから判断してほしい。
DATA 03
活動場所（沖縄県中頭郡西原町）
ホームは沖縄県中頭郡西原町。青い空とサトウキビ畑を眺めながらプレーする、最高のロケーションが自慢。
DATA 04
対戦募集（受付中）
VS申し込みは常時受付中。強豪チームも初心者チームも大歓迎。笑いの量が勝利の基準です。
特集 ひゃらら4大DNA　全貌解明

独占入手ひゃらら遺伝子の正体——4つのコアDNAを徹底分析

  • 全員卓球部出身
    サーブの回転はバッチリだが外角低めはお手上げ。卓球台の前では無敵、グラウンドでは修行中。この矛盾がひゃらら最大の魅力である。
  • 99年生まれで固めている
    1999年産の精鋭が集結。話が早く・ノリが合い・昭和ネタが通じる。同世代の結束力はプロ野球チーム以上かもしれない（個人の感想です）。
  • 西原町を愛している
    どこに遠征しても「早く西原帰りたい」と口にするメンバーたち。愛郷心のパラメータだけは間違いなくカンストしている。
  • 対戦相手を常に求めている
    DM文章の方がバットスイングより上手い疑惑あり。それでも全力で相手を求め続ける姿勢は、ある意味プロフェッショナルだ。
